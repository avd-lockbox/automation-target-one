# automation-target-one

Automation module for the target-one target application.

## Setup

```sh
# Install dependencies (requires CODEARTIFACT_AUTH_TOKEN)
yarn install

# Install git hooks
npx lefthook install
```

## Development

```sh
yarn build        # Compile TypeScript
yarn typecheck    # Type-check without emitting
yarn lint         # Run ESLint
yarn format       # Format with Prettier
yarn test         # Run Vitest tests
```
